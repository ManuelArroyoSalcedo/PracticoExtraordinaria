Gestión de Categorías

La gestión de categorías permite organizar los productos de forma estructurada, facilitando su clasificación y posterior búsqueda dentro del sistema.

Información que se almacena por cada categoría

Para cada categoría se registra la siguiente información:

  • Nombre: Denominación de la categoría (por ejemplo, "Electrónica", "Alimentación", "Papelería").
  • Descripción: Texto descriptivo que permite entender el tipo de productos que se incluyen en esta categoría.

Relaciones con otras entidades

Las categorías están vinculadas directamente con los productos. Cada producto debe pertenecer a una única categoría.

  • Productos: Una categoría puede estar asociada a múltiples productos.

Operaciones disponibles

Desde la interfaz de gestión de categorías se pueden realizar las siguientes operaciones:

  • Alta de categorías: Permite crear nuevas categorías especificando su nombre y descripción.
  • Modificación: Para actualizar el nombre o la descripción de una categoría existente.
  • Eliminación: Solo se puede eliminar una categoría si no tiene productos asociados.
  • Consulta: Visualización del listado de categorías registradas en el sistema.

Notas adicionales

Es recomendable utilizar nombres claros y descripciones concisas para facilitar la identificación de cada categoría. Una buena organización de las categorías contribuye a mejorar la usabilidad del sistema y la gestión eficiente de productos.
